Nathan's Cheet Sheet

Table Tags

Tag Name Description
<table> Table

The wrapper element for all HTML tables.
<thead> Table Head

The set of rows defining the column headers in a table.
<tbody> Table Body

The rows with all of the content of the table.
<tfooter> Table Foot

The set of rows defining the footer in a table.
<tr> Table Row

Creates a new Row
<td> Table Data

Adds a column of data.

Table Attributes

Tag Name Description
colspan Column Span

Spans the data across multiple columns.
rowspan Row Span

Spans the data across multiple rows.
